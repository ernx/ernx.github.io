About Cortex

We recognize that there is a challenge with knowledge sharing in a distributed organization. The Cortex bot is a Slack app that provides a quick method of sharing messages from within Slack and storing them to a designated target location, such as a Confluence page.

Usage

Installation

Install the Cortex app from the Slack app directory. If you don't have permissions to do so, seek assistance from your workspace admin.

Basic usage

Select the 3 vertical dots on a Slack message to open the actions menu. If you don’t see Cortex’s “Share Story” action, you may need to select “More message actions…” to find it from the available actions, as only your most used actions will appear in the context menu.

After selecting the “Share Story” message action, you will be presented with a modal dialog allowing you to enter a tagline, modify the message to be shared, and select a category.

Then press “Share” and the message will be automatically submitted to a register of stories on a designated Confluence page.

Alternative usage

Alternatively, you may trigger the “Share Story” screen in Slack by using a slash command as a starting point. Start a new message in Slack with the text /share and press enter. The Cortex Share Story screen should appear.
